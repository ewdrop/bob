# bob
daily lianxi

phase1 list:

​	day01:两数之和->空间换时间，根据当前数算得另一数，用哈希表查两数中另一数。时间复杂度O（n）。

​	day02：三树之和->先排序，固定外层值并取反为目标值，模拟左右指针，判断左右指针指向的两数和是否为取反后的目标值,时间复杂度O（n²）。

​	day03:无重复最长字串->哈希表存储判断未重复元素，发现重复元素更新索引值，每次算得当前无重复串的长度进行比较，时间复杂度O（n）。

​	day04:反转链表->一个临时节点用来保存后续节点地址，一个用来代替头节点，先这样再那样把指向反转，时间复杂度O（n）。

​	day05:合并两个有序链表->哑节点充当临时节点，判断两个链表的值，谁小指向谁，移动对应指针指向，如果有任一列表剩余直接合并到链表后。O(n+m)

​	day06：删除倒N节点->快慢指针，快指针先走N步，然后快慢指针同时走直到链表尾部，此时慢指针的后继则为要删除的节点，时间复杂度O（n）。

​	day07:链表节点两两交换->哑节点，first，second分别充当相邻两节点，交换指向后，移动当前节点为交换后的first，继续交换直到不存在可交换的节点，时间复杂度O（n）。

​	day08：环形链表->快慢指针，快指针一次两步慢指针一次一步，如果相遇，慢指针从头出发两指针一次一步再次相遇为环，时间复杂度O（n）。
